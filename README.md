Ecoarium 프로젝트는 사용자가 JT기기(수거 기기)를 통해 일회용 플라스틱 컵을 반납하여 스탬프를 적립받는 서비스입니다.
Ecoarium의 AP Server 파트는 node.js로 개발하여 사용자의 서비스 이용을 위한 웹 서버와 모바일 어플리케이션 서비스를 위한 API, 그리고 JT기기와 통신하는 API 및 머신러닝 모델을 포함하고 있습니다.<br><br>
